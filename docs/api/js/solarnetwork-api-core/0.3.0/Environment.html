Class: Environment

Environment

An environment configuration utility object.

This extends Configuration to add support for standard properties needed to access the SolarNetwork API, such as host and protocol values.

new Environment( [config])

Constructor.

This will define the following default properties, if not supplied on the config argument:

host
data.solarnetwork.net
protocol
https
port
443
